# PDF Toolbox Web

管理者権限なしで使える、ブラウザ起動型の PDF 統合・分割 / Web ファイル保存アプリです。ローカル PC 上で小さな Web サーバーを起動し、既定の Web ブラウザから操作します。

## 特徴

- ブラウザ上で PDF ファイルを複数選択して、指定順に 1 つの PDF に統合
- ブラウザ上で PDF を全ページ単位で分割、または指定ページ範囲ごとに分割して ZIP で保存
- Web ページ内のファイルリンク先データを、選択した形式だけ一括ダウンロード
- PDF、Word、Excel、画像、圧縮ファイルなどを複数選択して保存対象に指定可能
- インストール先を選ばないポータブル構成
- Windows の管理者権限なしで起動可能

## 必要環境

- Python 3.10 以降
- `pypdf`

## 管理者権限なしでのセットアップ

職場 PC などで管理者権限がない場合は、ユーザーフォルダー内に仮想環境を作成してください。

```bash
python -m venv .venv
.venv\\Scripts\\python -m pip install -r requirements.txt
.venv\\Scripts\\python pdf_toolbox.py
```

macOS / Linux の場合:

```bash
python3 -m venv .venv
./.venv/bin/python -m pip install -r requirements.txt
./.venv/bin/python pdf_toolbox.py
```

起動後、既定の Web ブラウザで `http://127.0.0.1:5000` が開きます。自動で開かない場合は、この URL をブラウザに入力してください。

## 使い方

### PDF の統合

1. ブラウザで「PDF 統合」を開きます。
2. 「統合する PDF」で統合したい PDF を複数選択します。
3. 「統合 PDF をダウンロード」を押すと、統合済み PDF がダウンロードされます。

### Web ファイル保存

1. ブラウザで「Web ファイル保存」を開きます。
2. ダウンロードしたいファイルリンクがある Web ページの URL を入力します。
3. PDF、Word、Excel、画像、ZIP など、保存したいファイル形式にチェックを入れます（複数選択可）。初期状態では PDF と Word が選択されています。
4. 「選択した形式のファイル ZIP をダウンロード」を押します。

選択した形式の直接ファイルリンクだけを ZIP にまとめて保存します。ブラウザーで現在開いているページは自動取得できないため、対象ページの URL をコピーして入力してください。

### PDF の分割

1. ブラウザで「PDF 分割」を開きます。
2. 「分割する PDF」で PDF を 1 つ選びます。
3. 分割方法を選びます。
   - 「1ページずつ分割」: 各ページを別々の PDF として ZIP に保存します。
   - 「ページ範囲で分割」: `1-3,5,8-10` のように入力した範囲ごとに PDF を作り、ZIP に保存します。
4. 「分割 PDF ZIP をダウンロード」を押します。

## 注意

パスワード保護された PDF や破損した PDF は処理できない場合があります。Web ファイル保存は、ページ内で直接ファイルを指すリンクのみ対象です。
